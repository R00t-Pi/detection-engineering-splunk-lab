# architecture

how telemetry moves from WIN10-01 to SOC01.

`architecture-v2.png` is the current diagram. If this ever needs a v3 (e.g. once AD or a
second endpoint gets added), keep the old version around with its own suffix rather than
overwriting it — it's useful to see how the lab actually grew, not just its current shape.
