# configs

The actual configuration files used to build this lab, not just a description of them —
so the setup in `docs/` can be verified or reproduced from something real instead of prose.

- `splunk/` — inputs.conf, outputs.conf, indexes.conf from SOC01
- `sysmon/` — the sysmonconfig.xml deployed on WIN10-01

Each subfolder has its own short README with the specifics.
